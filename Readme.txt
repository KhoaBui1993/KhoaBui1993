Appliance Business Management App

Requirement:

You need have the python evironment to run the application.
If you have any trouble to open the file, please make sure you install the python 3.0.
======================================================================================
Run the program:

Click on the Drive.py file,
If you want to test the database, please make sure to click the load_data_test to insert the data to table.
After that, click on the button: customer, employee and owner to see the customer_view, employee_view and owner_view of them.
